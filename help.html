MAP KMZ — 현장 기록관리 프로그램

KMZ 파일이란

KMZ는 구글어스·카카오맵 등에서 쓰는 지도 데이터 파일입니다. 마커(점), 선(트랙), 다각형 같은 위치 정보와 사진을 하나의 파일에 담습니다.

실은 ZIP 압축 파일입니다. 확장자를 .zip 으로 바꾸면 압축 프로그램으로 열 수 있고, 안에 위치정보를 담은 KML(XML) 파일과 사진(JPG)들이 그대로 들어 있습니다.

  • 사진을 KMZ 안에 함께 넣어 저장 → 위치 + 현장 사진을 파일 하나로 주고받기
  • 하나의 파일이라 이메일·메신저 전달, 백업이 편함
  • 휴대폰·PC 어디서든 열림 — 현장에서 찍고 사무실에서 확인

예전 방식의 문제 — 화면을 캡쳐해 이미지로 남기던 기록

  • 지도에 선을 그린 뒤 캡쳐해 문서에 붙여 넣고 나면, 그 작업을 어디에 저장해 뒀는지 찾기 어려움
  • 캡쳐 이미지에는 좌표가 없어, 원본을 못 찾으면 처음부터 다시 그려야 함
  • 시간이 지나면 어느 위치에서 무슨 작업을 했는지 기억조차 나지 않음

KMZ로 기록하면 해결됩니다

  • 다시 그릴 필요가 없습니다. 선·마커·사진이 좌표와 함께 저장되어, 파일을 지도에 끌어다 놓으면 작업 상태가 그대로 되살아나고 이어서 수정할 수 있습니다.
  • 어디에 뒀는지 기억 못해도 찾습니다. 제목·주소만으로 찾는 메타검색, 지도에서 지점을 찍어 찾는 위치로 검색이 있습니다.
  • 기억에 의존하지 않습니다. 촬영일시·GPS·제목·설명이 파일 안에 함께 남습니다.
  • 보고서가 자동으로 만들어집니다. 보고서 자동 작성 참고

그 밖의 이점

  • 사진의 GPS로 자동 배치 — 끌어다 놓기만 하면 촬영 지점에 표시
  • 모든 처리는 내 PC 안에서만 이루어지며 외부 서버로 전송되지 않습니다 (인터넷 없이도 동작, 지도 배경 타일 제외)
  • 엑셀 연동 — 복사한 위도·경도·주소를 엑셀에 붙여넣고, 엑셀 경로로 지도에 불러오기

화면 구성

  1. 지도 영역 — 마커·선·사진이 표시되는 곳. 우클릭으로 마커 추가, 휠로 확대/축소
  2. 주소 검색 — 주소나 장소 이름으로 지도 이동
  3. 도구 모음 — 로드뷰 · 도움말(F1) · 선 그리기(Alt) · 선 색상 · 선 두께 · 선 초기화 · 화면 초기화
  4. 저장 모음 — 저장 형식 · 저장 범위 · 사진 용량 · 사진명 변경 · 저장 · 지도 종류
  5. 목록 헤더 — 항목 개수와 Hold · 위치로 검색 · 필터 (목록 관리)
  6. 탭 필터 — 전체 / 사진 / 마커 / 트랙. 체크박스로 지도 표시를 켜고 끔
  7. 검색·목록 도구 — 메타검색 · 메타색인 (메타 검색), 삽입 · 복사 · 번호 · 이름 정리 · 항목표시
  8. 목록 표 — 번호 · 이름 · 주소 · 연장 · 촬영일시 · Geohash. 제목 클릭 시 정렬
  9. 메타정보 · 미리보기 — 선택 항목의 제목·설명·촬영일시(수정 가능), 사진 미리보기
  10. 좌표 · 주소 표시 — 클릭한 지점의 위도·경도·주소. 누르면 클립보드로 복사

사이드바와 지도 사이의 ◀ 버튼으로 목록을 접고 펼 수 있고, 경계선을 드래그해 너비를 조절합니다.

파일을 지도에 표시하는 방법

지원되는 파일 형식

  • KMZ / KML — 마커·선(트랙)·사진이 담긴 지도 파일
  • 사진 — JPG · JPEG · PNG · HEIC · HEIF · TIFF · TIF · WEBP · BMP
  • GPX — 트랙과 웨이포인트
  • CSV — 이름 · 위도 · 경도 · 주소 컬럼 자동 인식
  • HTM / HTML — 캐드 도면·웹지도의 선 데이터
  • ZIP — KMZ와 같은 구조의 압축 파일

방법 1. 지도창에 드래그 앤 드롭

  • 파일을 집어 지도 위에 끌어다 놓으면 바로 표시 — 연동 프로그램 없이도 동작
  • 여러 파일을 한 번에 놓을 수 있고, 이미 올려둔 것에 계속 더해집니다
  • 폴더 선택 버튼을 쓰면 하위 폴더까지 훑어 사진을 모두 불러옵니다
  • 이미 있는 파일을 다시 놓아도 중복으로 늘어나지 않습니다
  • Ctrl+V — 클립보드 이미지를 현재 지도 위치에 사진 마커로 붙여넣기

방법 2. 탐색기에서 파일을 선택하기만 하면 표시

연동 프로그램(사진위치표시)이 실행 중이면, 윈도우 탐색기에서 파일을 클릭해 선택하는 것만으로 지도에 나타납니다.

  • 임시 표시(흐리게) — 선택한 파일은 흐린 상태로 올라오고, 다른 파일을 고르면 자동으로 사라집니다. 여러 파일을 훑어볼 때 편리합니다.
  • 영구 확정 — 버튼, Insert 키, Ctrl+Shift+V 셋 중 하나로 확정합니다. 확정된 객체는 다른 파일을 선택해도 지워지지 않습니다.
  • Insert 키는 어디서든 동작 — 탐색기나 엑셀을 보는 중에도 지도창으로 갈 필요가 없습니다
  • 사진과 KMZ를 함께 선택하면 둘 다 표시됩니다
  • 지도의 사진을 Ctrl+C 하면 실제 사진 파일이 복사됩니다
  • GPS가 없는 사진을 탐색기에서 Ctrl+C → 지도 중심에 초록 마커로 놓입니다

방법 3. 엑셀 셀에 적힌 전체 경로를 선택하면 표시

엑셀 셀에 파일의 전체 경로(예: D:\현장\2026\장수읍_01.kmz)가 적혀 있으면, 그 셀을 클릭하는 것만으로 해당 파일이 지도에 표시됩니다. 현장 대장을 엑셀로 관리할 때 행을 따라 내려가며 위치를 바로 확인할 수 있습니다.

  • 사진 경로·KMZ 경로 모두 동작하며, KMZ 안에 든 사진 경로도 인식합니다
  • Ctrl+Shift+I — 경로가 적힌 셀 여러 개를 범위 선택한 뒤 누르면 한꺼번에 영구 삽입
  • 경로가 깨졌을 때 일괄 복구 — 워크북 전체 시트를 훑어 "없는 파일" 셀을 모두 찾고, 새 폴더를 한 번 지정하면 파일 이름으로 대조해 전부 새 경로로 고칩니다 (하위 폴더 검색, 수식 셀 제외). 못 찾은 파일이 남으면 다른 폴더를 다시 지정할 수 있고 [취소]로 종료합니다.
  • KMZ를 저장하면 전체 경로가 클립보드에 복사되어 엑셀에 바로 붙여넣을 수 있습니다

방법 2·3은 연동 프로그램이 실행 중이어야 합니다. 방법 1(드래그)은 프로그램 없이도 됩니다.

사진과 위치정보(GPS)

위치정보를 넣어 촬영하는 방법

  • 아이폰 — 설정 → 개인정보 보호 및 보안 → 위치 서비스 → 카메라 → "앱을 사용하는 동안" 허용 + "정확한 위치" 켜기
  • 안드로이드 — 카메라 앱 설정 → "위치 태그 / GPS 태그" 켜기, 위치 권한 허용
  • 실외에서 GPS 신호가 잡힌 뒤 촬영해야 정확합니다. 건물 안·지하는 오차가 큽니다

주의 — 카카오톡·메신저로 사진을 보내면 위치정보가 지워질 수 있습니다. "원본 보내기"를 사용하세요.

지도에서 사진 위치 보기

  • 사진을 지도에 드래그하면 GPS 위치에 사진 마커가 생깁니다
  • 목록에서 사진을 고르면 미리보기가 뜨고, 선택한 마커가 주황색으로 강조됩니다
  • 지도의 사진 마커를 클릭하면 목록에서도 자동 선택됩니다

사진 위치정보 수정 — 잘못 찍힌 위치 바로잡기

  1. 사진 마커를 1초간 길게 누릅니다 — 위치 이동 모드로 바뀝니다
  2. 원하는 위치로 끌어다 놓습니다 — 손을 떼면 확정되고 주소도 자동 갱신됩니다 (Esc 취소)
  3. 저장합니다 — 마커 클릭 후 💾 버튼으로 그 사진만, 또는 KMZ로 저장하면 안에 담기는 사진 전부에 반영

지도에서 옮긴 위치는 사진의 EXIF GPS 정보에 실제로 덮어쓰기됩니다. 그래서 다른 프로그램(꿀뷰·구글포토 등)이나 다른 PC에서 열어도 고친 위치가 그대로 유지됩니다.

위치정보가 없는 사진에 위치정보 넣기

GPS 없이 찍힌 사진도 지도에서 위치를 찍어 주면 위치정보가 들어간 새 사진을 얻을 수 있습니다.

  1. 지도창에 사진을 올립니다 — 위치를 알 수 없으므로 "지도 중심에 놓을까요?"라고 물어봅니다. 확인하면 지도 중심에 마커가 놓입니다. (탐색기에서 Ctrl+C 해도 동일)
  2. 마커를 실제 촬영 지점으로 옮깁니다 — 1초간 길게 눌러 이동 모드로 바꾼 뒤 끌어다 놓습니다. 위성지도를 켜고 건물·논밭 모양을 보며 맞추면 정확합니다.
  3. 저장합니다 — 💾 버튼 또는 KMZ 저장

저장된 사진에는 EXIF GPS 정보가 새로 기록되어, 위치정보가 없던 사진이 위치정보가 있는 사진으로 바뀝니다. 이후에는 드래그만 해도 촬영 지점에 표시되고, 보고서 자동 작성에도 그대로 쓸 수 있습니다.

오래된 사진, 위치 태그를 꺼두고 찍은 사진, 스캔한 사진에도 같은 방법으로 위치를 넣을 수 있습니다.

선(트랙) 그리기 · KMZ 저장

선 그리기

  • 툴바의 선 그리기(Alt) 버튼 또는 Alt 키로 시작/종료
  • 클릭할 때마다 점이 이어지며 거리가 실시간 표시됩니다. 우클릭하면 종료
  • Ctrl+Z / Backspace / Esc — 마지막 점 취소
  • 완성된 선을 우클릭(모바일은 길게 누르기)하면 수정·삭제

색상 · 두께 바꾸기

  • 색상 — 선 그리기 버튼 옆 색상 버튼을 누르면 팔레트가 열립니다
  • 두께 — 옆의 목록에서 1 · 2 · 3 · 5 · 8 · 12px 선택
  • 이미 그린 선 색 바꾸기 — 목록에서 트랙 왼쪽의 색상 막대를 클릭하면 팔레트가 열립니다
  • 선택·수정 중인 선은 원래 색을 유지한 채 조금 굵게 강조됩니다

KMZ로 저장 (사진도 함께 담김)

  • 저장 형식을 KMZ로, 저장 범위를 고른 뒤 💾 저장 또는 Ctrl+S
  • 마커·선·사진이 하나의 KMZ에 담깁니다 (ZIP 구조라 위치정보와 사진이 함께 압축)
  • 파일명은 읍면동명_날짜_마지막마커이름 으로 자동 생성 — 예: 산서백운리_20260422_현장A
  • 저장한 KMZ는 나중에 다시 드래그하면 그대로 복원됩니다
  • 저장 범위(전체객체 / 선택객체 / 각각저장)와 필터를 조합하면 원하는 것만 골라 저장할 수 있습니다

필요한 객체만 골라 새 KMZ로 저장하기

불러온 KMZ를 그대로 다시 저장할 필요는 없습니다. 필요 없는 것을 걸러 내고 원하는 객체만 담은 새 파일을 만들 수 있습니다.

  • 방법 1 — 지우고 저장 : 사이드바에서 필요 없는 객체를 Delete로 지우고, 저장 범위를 전체객체로 두고 저장
  • 방법 2 — 고른 것만 저장 : 지우지 않고, 저장할 객체만 Ctrl·Shift 클릭으로 선택한 뒤 저장 범위를 선택객체로 두고 저장
  • 방법 3 — 화면에 보이는 것만 저장 : 필터를 켜고 지도를 원하는 구역으로 맞춘 뒤 전체객체로 저장

원본 KMZ 파일은 그대로 남습니다. 저장은 항상 새 파일을 만드는 것이므로, 지우고 저장해도 원래 파일이 손상되지 않습니다.

여러 KMZ 파일을 하나로 합쳐 저장하기

여러 개로 나뉜 KMZ를 한 파일로 묶고 싶을 때 씁니다.

  1. 합칠 KMZ 파일들을 모두 지도에 올립니다 — 한꺼번에 끌어다 놓아도 되고 하나씩 올려도 됩니다. 새로 올린 것은 앞서 올린 것에 계속 더해집니다
  2. 필요 없는 객체가 있으면 사이드바에서 지웁니다 — 합치면서 정리까지 함께
  3. 저장 범위를 "전체객체"로 두고 KMZ로 저장합니다 — 올려 둔 모든 객체가 하나의 KMZ에 담기며, 각 파일에 들어 있던 사진도 함께 합쳐집니다
  • 현장별로 나눠 저장해 둔 KMZ들을 연도별·구역별로 하나로 묶어 보관할 때 편리합니다
  • 반대로 나누고 싶다면 저장 범위를 각각저장으로 두면 거리별로 여러 파일에 나뉘어 저장됩니다

사진 저장 옵션

사진 용량(압축) 조정

  • 100KB↓ / 200KB↓ / 300KB↓ / 500KB↓ / 800KB↓ / 1MB↓ — 그 용량 이하로 자동 압축
  • 원본 — 압축 없이 그대로 저장
  • 개별 다운로드, 전체 사진 다운로드, KMZ 안 사진 모두 이 설정이 적용됩니다. 용량을 줄이면 KMZ가 가벼워집니다

사진명 변경

  • 체크하면 저장되는 사진 파일명 뒤에 _000, _001 … 순서번호를 붙입니다
  • KMZ 안에 담기는 사진과 디스크의 원본 사진 모두에 적용됩니다
  • 목록 순서대로 번호가 매겨져 여러 장을 순서대로 정리할 때 편리합니다
  • 디스크의 원본 파일명까지 바꾸려면 연동 프로그램이 실행 중이어야 합니다

목록(사이드바) 관리 기능

객체 선택하기

  • 클릭 — 한 개 선택. 사진이면 미리보기가 뜨고 지도의 마커가 주황색으로 강조됩니다
  • Ctrl + 클릭 — 원하는 것만 하나씩 골라 담기 (다시 누르면 해제)
  • Shift + 클릭 — 먼저 고른 항목부터 지금 누른 항목까지 범위 선택
  • 드래그 — 여러 줄을 마우스로 쓸어 범위 선택
  • Ctrl + A — 현재 탭(전체/사진/마커/트랙)의 항목 모두 선택

필요 없는 객체 삭제 — 원하는 것만 남기기

  • 선택한 항목을 Delete 로 지웁니다. 목록에서 사라지고 지도에서도 함께 없어집니다
  • 잘못 지웠으면 Ctrl+Z 로 되돌릴 수 있습니다
  • 지도에서 선이나 마커를 우클릭해도 그 객체 하나를 바로 삭제할 수 있습니다
  • 지우지 않고 잠시 감추기만 하려면 탭 필터의 체크박스(사진·마커·트랙)를 끄면 됩니다

이렇게 정리한 뒤 저장하면 남아 있는 객체만 담긴 새 KMZ가 만들어집니다. 불러온 원본 KMZ는 그대로 남습니다. → 필요한 객체만 골라 저장하기

항목표시 — 표시할 컬럼 고르기

  • 목록에 표시할 항목을 체크로 고릅니다 — 주소 · 연장 · 촬영일시 · Geohash
  • 번호·이름은 항상 표시됩니다. 컬럼 제목을 클릭하면 그 값으로 정렬됩니다
  • 선택 상태는 저장되어 다음 실행 때도 유지됩니다

Geohash — 가까운 것끼리 정렬

  • Geohash는 위도·경도를 wydm6… 같은 짧은 문자열로 바꾼 위치 코드입니다 (8자리 ≈ ±19m)
  • 가까운 위치일수록 앞글자가 같습니다. 그래서 Geohash로 정렬하면 지리적으로 가까운 지점끼리 목록에서 자연스럽게 모입니다
  • 현장을 구역별로 묶어 보거나 이웃한 지점을 찾을 때 유용합니다

필터 — 화면에 보이는 것만

  • 켜면 지금 지도 화면에 보이는 객체만 목록에 나타납니다 (지도를 움직이면 실시간 갱신)
  • 필터를 켠 채 "전체객체"로 저장하면 화면 안에 보이는 객체만 저장됩니다

Hold — 지도 안 움직이게 고정

  • 켜면 목록이나 검색 결과에서 객체를 골라도 지도 화면이 이동하지 않습니다
  • 지금 보는 위치를 유지한 채 여러 항목을 훑어볼 때 켭니다

위치로 검색

  • 누른 뒤 ① 지도에서 중심점 클릭 → ② 마우스를 움직여 반경을 정하고 다시 클릭
  • 그 원 안에 있는 KMZ·사진을 폴더(하위 포함)에서 찾아 줍니다
  • "이 근처에서 예전에 찍은 자료가 있었나?"를 위치로 되짚어 찾을 때 사용합니다 (Esc 취소)

메타색인 · 메타검색 · 메타정보 입력

메타정보란 사진의 제목·설명·촬영일시처럼 파일에 딸린 부가 정보입니다.

메타정보 입력

  • 사이드바에서 사진을 고르면 메타 패널에 제목·설명·촬영일시가 나옵니다
  • 수정 버튼으로 제목·설명을 입력해 저장하면 사진의 EXIF에 실제로 기록됩니다 (꿀뷰 등 다른 뷰어에서도 보임)
  • 이름은 목록에서 더블클릭(또는 선택 후 Enter)해 바로 바꿀 수 있습니다

메타색인 — 검색되게 등록해두기

  • 폴더를 지정하면 그 안 JPG·KMZ의 메타정보를 미리 색인에 등록합니다
  • 한 번 색인한 폴더는 자동 감시됩니다 — 이후 파일을 추가·수정·삭제하면 자동으로 색인이 갱신되어 다시 색인할 필요가 없습니다 (폴더 밖은 감시하지 않음)

메타검색 — 색인에서 즉시 찾기

  • 검색어를 넣으면 JPG(제목·설명·촬영일시), KMZ(항목 정보), KMZ 안에 든 사진까지 즉시 찾아 줍니다
  • 필드를 지정할 수 있습니다 — 예: 제목:정자나무, 주소:강남
  • 여러 조건을 공백으로 나열하면 모두 만족(AND)하는 것만 찾습니다
  • 어느 폴더에 있는지 몰라도 제목·주소만으로 바로 찾아낼 수 있습니다

보고서 자동 작성 곧 공개 예정

위치정보가 있는 사진이 있거나, 지도에서 KMZ 파일을 만들어 두면 그것을 바탕으로 현장 보고서를 자동으로 작성할 수 있습니다.

어떻게 동작하나

  • 사진에 들어 있는 위치·촬영일시와 지도에서 조회한 주소, 입력해 둔 제목·설명이 그대로 보고서 항목이 됩니다
  • KMZ에 담아 둔 마커·선(트랙)·사진도 같은 방식으로 보고서에 옮겨집니다
  • 따로 표를 만들거나 사진을 하나씩 붙여 넣을 필요가 없습니다. 현장에서 사진만 제대로 찍어 오면 보고서의 뼈대가 자동으로 채워집니다

준비해 두면 좋은 것

  • 사진에 위치정보를 넣어 촬영하기
  • 위치정보가 없는 사진은 지도에서 위치를 지정해 저장해 두기
  • 사진의 제목·설명을 미리 입력해 두기 — 보고서에 그대로 실립니다
  • 현장 단위로 KMZ를 저장해 두기 — 나중에 그 파일 하나만 열면 보고서를 다시 만들 수 있습니다

이 기능은 현재 준비 중이며 곧 공개할 예정입니다. 지금 찍어 두는 사진과 저장해 두는 KMZ는 공개 후 그대로 사용할 수 있으니, 위 내용대로 자료를 모아 두시면 됩니다.
