개요

MAP KMZ는 현장 위치를 기록하고 파일로 저장할 수 있는 도구입니다. 저장한 파일을 지도에 드래그해서 올려놓으면 언제든지 다시 확인할 수 있습니다. 마커와 선으로 현장 정보를 표시하고, KMZ / GPX / CSV 형식으로 저장하거나 불러올 수 있습니다.

📁 지원 파일 형식

  • 불러오기: KMZ, ZIP, GPX, JPG / PNG / HEIC (사진), CSV, HTM
  • 내보내기: KMZ, GPX, CSV, 사진 다운로드

✏️ 불러온 파일 수정

불러온 KMZ 파일의 마커나 선을 우클릭하면 수정·삭제 메뉴가 나타납니다. 수정 후 다시 저장하면 업데이트된 파일을 만들 수 있습니다.

🔒 개인정보 보호 안내 — 모든 처리는 내 PC에서만 이루어집니다
사진 드래그, GPS 정보 읽기, 위치 수정, 사진 압축, 저장 등 모든 작업은 사용자의 브라우저(PC) 안에서만 처리됩니다. 사진 파일은 외부 서버로 전송되거나 업로드되지 않으며, 인터넷 연결 없이도 동작합니다 (지도 타일 제외).
📂

파일 불러오기

🖱️ 드래그 앤 드롭

아래 파일 형식을 지도 위로 끌어다 놓으면 자동으로 불러옵니다. 여러 파일을 한 번에 드롭할 수 있으며, 기존 데이터에 추가되는 방식으로 불러와집니다.

  • KMZ / ZIP: 마커·선·사진 포함 지도 파일
  • GPX: 트랙·웨이포인트 파일
  • HTM / HTML: 카카오맵 기반 웹 지도 파일 (linePath, polygonPath, positions 형식 지원)
  • CSV: 마커 목록 파일 (이름·위도·경도·주소 컬럼 자동 감지)
  • JPG / PNG / HEIC: 사진 파일 (GPS 있으면 촬영 위치에 자동 배치)

📁 폴더 선택 (사진 일괄 추가)

  • 사이드바 상단의 📁 폴더 선택 버튼을 클릭합니다
  • 선택한 폴더 안의 모든 사진 파일이 자동으로 추가됩니다
  • 하위 폴더까지 재귀적으로 탐색하여 포함합니다
  • GPS 정보가 있는 사진은 촬영 위치에, 없는 사진은 선택 시 지도 중심에 배치됩니다
  • Chrome / Edge 브라우저에서만 지원됩니다

📋 클립보드 붙여넣기 (Ctrl+V)

클립보드에 복사된 이미지를 Ctrl+V로 붙여넣으면 현재 지도 위치에 사진 마커로 추가됩니다.

팁: 휴대폰에서 저장된 KMZ 파일은 .kmz 형식으로 저장됩니다. PC로 옮긴 후 지도에 드래그해서 불러올 수 있습니다.
📍

마커

➕ 마커 추가

  • 지도를 우클릭하면 마커 이름 입력창이 나타납니다
  • 이름 입력 후 추가 버튼 또는 Enter로 생성
  • 현장 위치마다 마커를 추가해 기록할 수 있습니다

✏️ 마커 수정 / 삭제

  • 마커를 클릭하면 이름이 표시됩니다
  • 마커를 우클릭하면 수정 / 삭제 메뉴가 나타납니다
  • 불러온 파일의 마커도 동일하게 수정·삭제할 수 있습니다
  • 마커를 클릭하면 사이드바 목록에서도 해당 항목이 선택됩니다

🔀 마커 위치 이동

  • 마커를 1초간 길게 누르면 위치 이동 모드가 시작됩니다
  • 마우스를 드래그해 원하는 위치로 이동 후 손을 떼면 확정됩니다
  • 이동 후 사이드바 주소가 자동으로 업데이트됩니다
  • 이동된 위치로 KMZ/사진 저장 시 새 위치의 GPS 정보가 반영됩니다
  • Escape: 이동 취소

🔢 위치 복사 마커 (번호 마커)

  • 주소 클릭 또는 Ctrl+C를 누르면 현재 위치의 위도·경도·주소가 클립보드에 저장되고
  • 해당 위치에 1, 2, 3... 순서로 번호 마커가 자동 생성됩니다
  • 화면 초기화 후에는 번호가 1부터 다시 시작합니다
✏️

선 그리기

🖊️ 선 그리기 시작 / 종료

  • 선 그리기 버튼 클릭 또는 Alt 키로 시작 / 종료
  • 지도를 클릭할 때마다 포인트가 추가되며 거리가 실시간으로 표시됩니다
  • 우클릭으로 선 그리기 종료

↩️ 이전 포인트 삭제

Ctrl+Z / Backspace / Escape로 마지막 포인트를 삭제합니다.

🔧 완성된 선 수정 / 삭제

  • 완성된 선을 우클릭 (PC) 또는 길게 누르기 (모바일)
  • 수정: 기존 포인트를 유지한 채 그리기 모드로 복원하여 계속 이어 그릴 수 있습니다
  • 삭제: 선 제거
  • 불러온 KMZ 파일의 선도 동일하게 수정·삭제할 수 있습니다
📸

사진

🔒 사진은 서버에 업로드되지 않습니다
사진을 지도에 드래그하거나 폴더에서 불러와도 파일은 외부로 전송되지 않습니다. GPS 정보 추출, 위치 표시, 압축, 저장 등 모든 처리는 사용자의 PC 브라우저 안에서만 이루어집니다. 인터넷 차단 환경에서도 사진 관련 기능은 정상 동작합니다.

📷 사진 불러오기

  • JPG / PNG / HEIC 파일을 지도에 드래그 앤 드롭으로 추가
  • 사진에 GPS 정보가 있으면 촬영 위치에 마커가 자동 생성됩니다
  • GPS 정보가 없으면 현재 지도 중심에 추가 여부를 묻습니다
  • 📁 폴더 선택 버튼으로 하위폴더까지 한 번에 불러올 수 있습니다

🔧 사진 위치 수정 후 저장

  • 사진 마커를 1초간 길게 눌러 원하는 위치로 드래그하면 GPS 위치 정보가 수정됩니다
  • 사진 마커 클릭 후 💾 버튼: 수정된 위치가 EXIF GPS에 반영된 사진을 저장
  • KMZ 저장 시에도 이동된 위치의 GPS 정보가 사진 내부에 자동 반영됩니다
  • GPS가 없는 사진에 위치를 직접 지정하거나 잘못된 위치를 수정할 때 활용할 수 있습니다

🎚️ 사진 용량 선택

  • 저장 버튼 옆 선택 목록에서 저장 용량을 선택합니다
  • 200KB↓ / 300KB↓ / 500KB↓ / 800KB↓ / 1MB↓: 해당 용량 이하로 압축 저장
  • 원본: 압축 없이 원본 그대로 저장
  • 개별 사진 다운로드, 전체 사진 다운로드, KMZ 내 사진 모두 이 설정이 적용됩니다

💾 사진 다운로드

  • 사진 마커 클릭 후 💾 버튼으로 개별 다운로드
  • 저장 형식을 📸 사진 다운로드로 선택 후 저장 버튼: 모든 사진을 한 번에 다운로드
  • 저장 영역을 선택 객체로 설정하면 사이드바에서 선택한 사진만 다운로드

목록 패널 (사이드바)

📂 목록 열기 / 닫기

  • 지도 왼쪽의 ▶ 버튼으로 목록 패널을 열고 닫습니다
  • 마커·사진·트랙을 처음 추가하면 목록 패널이 자동으로 열립니다
  • 패널 오른쪽 경계선을 드래그해 너비를 조절할 수 있습니다
  • 미리보기 영역 위 경계선을 드래그해 높이를 조절할 수 있습니다

🗂️ 탭 필터

  • 전체 / 사진 / 마커 / 트랙 탭으로 목록을 필터링합니다
  • 각 탭에 현재 항목 수가 표시됩니다

🖱️ 항목 선택 및 지도 이동

  • 항목 클릭: 해당 마커/트랙 위치로 지도가 이동하고 팝업이 표시됩니다
  • 사진 항목 선택 시 하단 미리보기 창에 사진이 표시됩니다
  • 지도에서 마커/사진을 클릭하면 목록에서 해당 항목이 자동 선택됩니다
  • 목록 빈 공간 클릭 시 선택이 모두 해제됩니다

☑️ 다중 선택

  • Ctrl + 클릭: 개별 항목을 추가/제거하며 선택
  • Shift + 클릭: 마지막 선택 항목부터 현재 항목까지 범위 선택
  • 드래그: 여러 행을 마우스로 쓸어 범위 선택
  • Ctrl + A: 현재 탭의 모든 항목 선택

✏️ 이름 수정

  • 항목 이름을 더블클릭하면 이름 수정 모드가 시작됩니다
  • 목록에서 항목 선택 후 Enter를 누르면 이름 수정 모드가 시작됩니다
  • Enter: 수정 완료 / Escape: 취소
  • 다른 항목을 클릭하면 자동으로 저장됩니다

⌨️ 키보드 조작

  • ↑ / ↓ 방향키: 목록에서 이전/다음 항목으로 이동 (지도도 함께 이동)
  • Shift + ↑/↓: 이동하면서 연속 선택
  • Enter: 선택된 항목 이름 수정 모드
  • Delete: 선택된 항목 삭제 (확인 후 삭제)
  • Ctrl + A: 전체 선택

위치 복사

📍 위도·경도 표시

화면 상단의 좌표 표시 창에 현재 클릭한 위치의 위도, 경도가 표시됩니다. 클릭하면 좌표가 클립보드에 복사됩니다.

📋 위도·경도·주소 한 번에 복사

  • 화면의 주소를 클릭하거나 Ctrl+C를 누르면
  • 위도   경도   주소 형식으로 클립보드에 저장됩니다
  • 엑셀에 붙여넣으면 세 칸에 자동으로 나뉘어 입력됩니다
  • 복사와 동시에 해당 위치에 번호 마커(1, 2, 3...)가 자동 생성됩니다
  • 화면 초기화 후에는 번호가 1부터 다시 시작합니다
💾

내보내기

📤 저장 방법

  • 파일 형식 선택 → 저장 영역 선택 → 💾 저장 버튼 또는 Ctrl+S
  • 전체 영역: 지도에 있는 모든 마커·선 저장
  • 현재 영역: 현재 화면에 보이는 데이터만 저장
  • 선택 객체: 사이드바 목록에서 선택한 항목만 저장

☑️ 선택 객체 저장

  • 사이드바에서 원하는 항목을 선택(단일·다중 선택 모두 가능)한 후
  • 저장 영역을 선택 객체로 설정하고 저장하면 해당 항목만 내보냅니다
  • KMZ, GPX, CSV, 사진 다운로드 모두 선택 객체 저장을 지원합니다

📦 KMZ 저장

  • 마커·선·사진을 모두 포함하여 저장합니다
  • 파일명은 읍면동명 + 날짜 + 마지막 마커이름으로 자동 생성됩니다
  • 예: 산서백운리_20260422_현장A
  • 저장된 KMZ 파일을 나중에 지도에 드래그해서 다시 불러올 수 있습니다
  • 사진은 자동 압축되며, 마커 이동 후 저장 시 이동된 위치의 GPS가 사진에 반영됩니다

🗺️ GPX 저장

선(트랙) 데이터와 마커(웨이포인트)가 GPX 형식으로 저장됩니다.

📊 CSV 저장

  • 마커 데이터를 CSV로 저장합니다
  • 컬럼: 이름, 위도, 경도, 주소
  • 엑셀에서 바로 열 수 있습니다 (한글 깨짐 없음)
  • 저장한 CSV 파일을 지도에 드래그해서 마커로 불러올 수 있습니다
