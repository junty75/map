개요

MAP KMZ는 현장 위치를 기록하고 파일로 저장할 수 있는 도구입니다. 저장한 파일을 지도에 드래그해서 올려놓으면 언제든지 다시 확인할 수 있습니다. 마커와 선으로 현장 정보를 표시하고, KMZ / GPX / CSV 형식으로 저장하거나 불러올 수 있습니다.

📁 지원 파일 형식

  • 불러오기: KMZ, ZIP, GPX, JPG / PNG (사진)
  • 내보내기: KMZ, GPX, CSV

✏️ 불러온 파일 수정

불러온 KMZ 파일의 마커나 선을 우클릭하면 수정·삭제 메뉴가 나타납니다. 수정 후 다시 저장하면 업데이트된 파일을 만들 수 있습니다.

📂

파일 불러오기

🖱️ 드래그 앤 드롭

KMZ, GPX, 사진 파일을 지도 위로 끌어다 놓으면 자동으로 불러옵니다. 여러 파일을 한 번에 드롭할 수 있으며, 기존 데이터에 추가되는 방식으로 불러와집니다.

📋 클립보드 붙여넣기 (Ctrl+V)

클립보드에 복사된 이미지를 Ctrl+V로 붙여넣으면 현재 지도 위치에 사진 마커로 추가됩니다.

팁: 휴대폰에서 저장된 KMZ 파일은 .kmz 형식으로 저장됩니다. 파일을 PC로 옮긴 후 지도에 드래그해서 불러올 수 있습니다.
📍

마커

➕ 마커 추가

  • 지도를 우클릭하면 마커 이름 입력창이 나타납니다
  • 이름 입력 후 추가 버튼 또는 Enter로 생성
  • 현장 위치마다 마커를 추가해 기록할 수 있습니다

✏️ 마커 수정 / 삭제

  • 마커를 클릭하면 이름이 표시됩니다
  • 마커를 우클릭하면 수정 / 삭제 메뉴가 나타납니다
  • 불러온 파일의 마커도 동일하게 수정·삭제할 수 있습니다

🔢 위치 복사 마커 (번호 마커)

  • 주소 클릭 또는 Ctrl+C를 누르면 현재 위치의 위도·경도·주소가 클립보드에 저장되고
  • 해당 위치에 1, 2, 3... 순서로 번호 마커가 자동 생성됩니다
  • 화면 초기화 후에는 번호가 1부터 다시 시작합니다
✏️

선 그리기

🖊️ 선 그리기 시작 / 종료

  • 선 그리기 버튼 클릭 또는 Alt 키로 시작 / 종료
  • 지도를 클릭할 때마다 포인트가 추가되며 거리가 실시간으로 표시됩니다
  • 우클릭으로 선 그리기 종료

↩️ 이전 포인트 삭제

Ctrl+Z / Backspace / Escape로 마지막 포인트를 삭제합니다.

🔧 완성된 선 수정 / 삭제

  • 완성된 선을 우클릭 (PC) 또는 길게 누르기 (모바일)
  • 수정: 기존 포인트를 유지한 채 그리기 모드로 복원하여 계속 이어 그릴 수 있습니다
  • 삭제: 선 제거
  • 불러온 KMZ 파일의 선도 동일하게 수정·삭제할 수 있습니다
📸

사진

📷 사진 불러오기

  • JPG / PNG 파일을 지도에 드래그 앤 드롭으로 추가
  • 사진에 GPS 정보가 있으면 촬영 위치에 마커가 자동 생성됩니다
  • GPS 정보가 없으면 현재 지도 중심에 추가 여부를 묻습니다

🔧 사진 위치 수정 후 다운로드

  • 사진 마커를 원하는 위치로 드래그하면 GPS 위치 정보가 수정됩니다
  • 사진 마커 클릭 후 다운로드 버튼을 누르면 수정된 위치가 반영된 사진을 저장할 수 있습니다
  • GPS가 없는 사진에 위치를 직접 지정하거나 잘못된 위치를 수정할 때 활용할 수 있습니다

💾 사진 다운로드

  • 사진 마커 클릭 후 개별 다운로드 가능
  • 사진 다운로드 버튼으로 모든 사진을 한 번에 다운로드
팁: KMZ 저장 시 사진은 자동으로 300KB 이하로 압축됩니다.
📌

위치 복사

📋 클립보드 복사

  • 화면 상단의 주소를 클릭하거나 Ctrl+C를 누르면
  • 위도   경도   주소 형식으로 클립보드에 저장됩니다
  • 엑셀에 붙여넣으면 세 칸에 자동으로 나뉘어 입력됩니다
  • 복사와 동시에 해당 위치에 번호 마커가 자동 생성됩니다
💾

내보내기

📦 KMZ 저장

  • 저장 버튼: 전체 마커와 선을 KMZ로 저장
  • Ctrl+S: 현재 화면에 보이는 데이터만 저장
  • 파일명은 읍면동명 + 날짜 + 마지막 마커이름으로 자동 생성됩니다
  • 예: 산서백운리_20260422_현장A
  • 저장된 KMZ 파일을 나중에 지도에 드래그해서 다시 불러올 수 있습니다

🗺️ GPX 저장

형식 선택 후 저장 버튼 또는 Ctrl+S로 내보냅니다. 선(폴리라인) 데이터가 GPX 트랙으로 저장됩니다.

📊 CSV 저장

  • 마커 데이터를 CSV로 저장합니다
  • 컬럼: 번호, 이름, 위도, 경도, 주소, 사진경로
  • 엑셀에서 바로 열 수 있습니다 (한글 깨짐 없음)

모바일 앱

📲 크롬 브라우저에서 앱 설치

  • 휴대폰 크롬 브라우저에서 아래 주소로 접속합니다
  • https://junty75.github.io/img
  • 오른쪽 상단의 설치 버튼을 눌러 앱을 설치합니다
  • 설치 후 홈 화면에서 일반 앱처럼 실행할 수 있습니다

📱 모바일에서 할 수 있는 것

  • PC에서 저장한 KMZ 파일을 휴대폰에서 열어 확인할 수 있습니다
  • 현장에서 GPS 위치를 확인하며 마커와 선을 기록할 수 있습니다
  • 휴대폰에서 저장하면 .kmz 형식으로 저장됩니다
  • 저장된 파일을 PC 지도에 드래그해서 불러올 수 있습니다

📲 모바일 조작 방법

  • 길게 누르기: PC의 우클릭과 동일 (마커 추가, 선 수정/삭제)
  • 뒤로가기 버튼: 열려있는 창 닫기 → 선 그리기 포인트 삭제 → 앱 종료 순서
